水箱液位显示 15（冷水箱和热水箱需要切换显示，使用数字量开关做控制
				当数字输出为0时，该通道采集数据为冷水箱的液位高度
				当数字输出为1时，该通道采集数据为热水箱的液位高度
				并且增加两个控制冷热水箱的进水阀，具体的数字量io详情见下面数字量的通道详情。
				）
tip：水箱的液面需要设置上下限，0~10v对应0~2000mm的高度。

当水箱的液位低于下限，打开对应水箱的进水阀
当水箱的液位超过上限，关闭对应水箱的进水阀
因此，需要在系统设置中增加两个水箱液面的上下限的设置

数据采集逻辑：对应的水箱液面信息均在切换的时候采集，

EX：当前开关状态是0，当延时5s后切换为1，在切换之前采集对应开关是0的时候冷水箱的液位高度数据并进行判断，做出对应的行为。
	同理，当开关状态切换为1后，延时5s后切换为0，在切换之前采集对应开关是1的时候热水箱的液位高度数据并进行判断，做出对应的行为

=========================新增（end）====================
